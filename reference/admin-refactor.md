## 관리자 페이지 변경

### 메인페이지의 헤더와 푸터는 보이지 않도록
- 지금 메인페이지에서 사용 중인 헤더와 푸터는 보이지 않아야 한다.

### 관리자 전용 페이지 세분화 
지금은 제조사, 판매차량, 출고차량이 모두 admin 페이지에 있는데
각각 별도의 페이지로 분리해야한다.

### 관리자 화면 헤더 구성요소
좌에서부터 오른쪽으로 배치
- logo.png (메인페이지와 동일하게) (누르면 메인페이지로 이동)
- 제조사 버튼 -> 누르면 제조사 관리 페이지로
- 판매차량 버튼 -> 누르면 판매차량 관리페이지로 
- 출고차량 버튼 -> 누르면 출고차량 관리페이지로
- 로그아웃 버튼

### 등록, 수정 관련 페이지 별도 생성
- 타겟
  - 제조사
  - 판매차량
  - 출고차량
- 목록페이지, 등록 페이지와 수정페이지 별도 존재

### 목록페이지 구성요소
- 목록 하단에 페이지네이션 배치
- 각 목록 row 마다 (제목, 작성자 닉네임, 작성날짜 '2025-01-23' 형식)

### 상세페이지 구성요소
- 각 데이터별 정보를 보여준다.
- 수정 버튼, 삭제 버튼 존재

### 테마 색상 변경
- 블루 계열 색상이 아니라 메탈 계열 색상을 사용
- globals.css에 있는 primary, secondary 색상을 최대한 활용
